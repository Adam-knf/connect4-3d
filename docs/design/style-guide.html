07. 胜负特效

胜利特效

胜利
连线高亮（绿色发光）→ 棋盘旋转 → 金色粒子爆炸

失败特效

失败
连线高亮（红色发光）→ 灰色粒子飘落 → 结果文字

特效规格

  • ◈
    连线高亮
    获胜4子连线使用 emissive 发光材质，颜色随胜利/失败变化
  • ✧
    粒子系统
    胜利：金色粒子向外爆炸扩散；失败：灰色粒子向下飘落
  • ⟳
    棋盘旋转
    胜利时棋盘自动旋转360度展示连线，失败时不旋转
